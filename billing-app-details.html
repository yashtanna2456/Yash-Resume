BillPro - Desktop Billing App

A comprehensive, dual-mode invoicing and billing management system developed as a hybrid desktop application. Designed to serve both standard company invoicing and specialized shop invoicing with a full suite of tools for invoice creation, client management, payment tracking, and business analytics.

Project Overview

BillPro is a comprehensive, dual-mode invoicing and billing management system developed as a hybrid desktop application. It is designed to serve two distinct user needs: a standard "Company" invoicing system for managing clients and services, and a specialized "Shop" invoicing system for retail or point-of-sale environments. The application provides a full suite of tools for invoice creation, client management, payment tracking, and business analytics. By leveraging pywebview, it packages a powerful Django backend and a dynamic JavaScript frontend into a single, seamless desktop experience, offering the power of a web application with the convenience of a native app.

Technical Implementation

Architecture

Django MVT Pattern: Built on Django's Model-View-Template architecture ensuring clean separation of concerns between data logic, business logic, and presentation layer.

Database Modeling

Relational Schema: Designed using Django ORM to manage Users, Clients, Invoices, InvoiceItems, and InvoiceTemplates. A boolean flag (is_shop_invoice) effectively segregates the two application modes.

Template Customization Engine

The invoice designer's frontend serializes layout data (element positions, sizes, styles, custom table headers) into a single JSON object. This JSON is saved to the template_data field in the InvoiceTemplate model. Preview and PDF generation views parse this JSON to dynamically construct HTML/CSS for pixel-perfect rendering.

Desktop Bridge

A Python-to-JavaScript API was created using pywebview's js_api functionality. This bridge allows the frontend to securely call Python functions, enabling features like saving/clearing encrypted credentials for auto-login without exposing sensitive logic to the client-side.

Data-Driven Analytics

Business insights views perform complex database aggregations (Sum, Count, Avg) using Django ORM. Data is serialized into JSON and passed to the frontend where Chart.js renders interactive visualizations.
