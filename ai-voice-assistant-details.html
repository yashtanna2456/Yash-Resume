Kiara AI - Voice Assistant

A comprehensive voice-activated personal assistant built in Python, designed to interact with users through speech recognition and text-to-speech capabilities, performing various system tasks and providing information through voice commands.

Project Overview

Kiara AI is a comprehensive voice-activated personal assistant built in Python. It's designed to interact with users through speech recognition and text-to-speech capabilities, performing various system tasks and providing information through voice commands. The assistant greets users based on the time of day and maintains a conversational interface throughout the interaction.

Key Features

Core Functionality

  • Voice Recognition: Converts speech to text using Google's speech recognition API
  • Text-to-Speech: Responds with natural voice output using pyttsx3
  • Time-based Greetings: Contextual greetings based on current time
  • Continuous Listening: Runs in a loop to continuously accept commands

System Control

  • Volume Management: Increase/decrease system volume by 10% increments
  • Mute/Unmute: Toggle system audio mute state
  • Application Launching: Open Chrome, VS Code, Downloads folder
  • Screenshot Capture: Take and save screenshots as "ss.jpg"
  • Camera Control: Automated camera app interaction for photos

Web & Information Services

  • Wikipedia Search: Query and read Wikipedia summaries (2 sentences)
  • Web Browsing: Open YouTube, Google, and custom websites
  • General Search: Perform web searches using pywhatkit
  • Weather Access: Redirect to weather.com for weather information

Entertainment & Communication

  • Music Playback: Play local music files from specified directory
  • Joke Telling: Random joke selection from predefined collection
  • Email Sending: Send emails via Gmail SMTP (to specific contact)
  • Conversational Responses: Handle greetings, thanks, and casual conversation
